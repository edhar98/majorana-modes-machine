\begin{frame}{The Observable We Keep}

  \begin{equation*}
    O_{\mathrm{edge}} = \Ostring
  \end{equation*}

  \begin{itemize}
    \item Single-edge operators flip fermion parity, so their expectation vanishes in a parity eigenstate.

    \item Unlike Ising-type order parameters, topology cannot generally be detected locally.

    \item The Jordan--Wigner mapping converts an edge-to-edge Majorana correlator into a non-local Pauli string.

    \item The two-edge string preserves parity and probes coherent long-range edge correlations.

    \item For $L=4$:
    \[
      O_{\mathrm{edge}} = X_0 Z_1 Z_2 X_3.
    \]
  \end{itemize}

\end{frame}

\begin{frame}{Measurement Protocol}

  \begin{columns}[T]

    \begin{column}{0.52\textwidth}
      \textbf{Circuit-level measurement}

      \begin{itemize}\small
        \item Current validation: exact even-parity ground state from diagonalization.
        \item Hardware target: parity-constrained VQE state preparation.
        \item Apply Hadamards on qubits carrying $X$ factors.
        \item Measure all qubits and multiply outcomes to estimate $\langle O_{\mathrm{edge}}\rangle$.
      \end{itemize}
    \end{column}

    \begin{column}{0.44\textwidth}

      \begin{block}{Shot estimator ($N = 4096$)}\small

        Each shot returns an eigenvalue $m_s = \pm 1$.

        \[
          \langle O\rangle \approx \frac{1}{N}\sum_{s=1}^{N} m_s,
          \quad
          \sigma \approx \sqrt{\frac{1-\langle O\rangle^2}{N}}
        \]

        For any $\pm1$ estimator with $N=4096$, $\sigma\leq 1/\sqrt{N}\approx0.016$. Error bars are omitted from the sweep plot for clarity.

      \end{block}

    \end{column}

  \end{columns}

\end{frame}

\begin{frame}[fragile]{Implementation Step}

  \begin{lstlisting}[basicstyle=\ttfamily\scriptsize]
for mu in mu_values:
    H = kitaev_qubit_hamiltonian(L, t, mu, delta)
    psi = even_parity_ground_state(H)   # exact diagonalization, NOT VQE
    value = <psi| X0 Z1 ... Z(L-2) X(L-1) |psi>
    shot_value = sample_pm_one(value, shots=4096)
  \end{lstlisting}

  \begin{itemize}

    \item \textbf{Note:} \texttt{even\_parity\_ground\_state} uses exact diagonalization -- not VQE. The pseudocode masks this gap; a real VQE loop is the next milestone.

    \item Exact diagonalization is valid only for small $L$; it serves as a validation benchmark, not a scalable method.

    \item The measured quantity is the same Pauli string a circuit would estimate from bitstrings.

    \item The same workflow can be executed on quantum hardware using repeated circuit measurements.

    \item The next code revision should plug this sweep into the VQE runner directly.

  \end{itemize}

\begin{frame}{Main Conclusions}

  \begin{itemize}
    \item The edge Pauli string is a measurable finite-size diagnostic of Kitaev topology.
    \item Exact diagonalization shows string enhancement alongside near-degenerate parity sectors.
    \item Synthetic shot sampling reproduces the ideal string-correlation trend.
    \item A real VQE sweep and hardware-noise study remain the next validation steps.
  \end{itemize}

  \vspace{0.4cm}

  \begin{block}{Conceptual Message}
    Local observables may change across the transition, but parity-sector
    degeneracy and non-local edge coherence identify the topological phase.
  \end{block}

\end{frame}
